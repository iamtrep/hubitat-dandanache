# Zigbee OTA images

Images are stored in the `images` directory. To get details about a specific image file:

```sh
node info.js images/xxx.zigbee
```

## IKEA - 117C
 To download Zigbee OTA files for IKEA devices, run the following command:

 ```sh
 node ikea.js
 ```
